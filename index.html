Create your Three.js Binary Animation(TBA) file. It is a binary file generated from retargeting animation, with only ~16% the size of a JSON file.

How to Convert Animations

To convert your animation, follow these steps:

  • 1. Drag and drop the Source file (the animation FBX exported from Mixamo).
  • 2. Drag and drop the Target file (the character GLB exported from Ready Player Me). (Optional)
  • 3. Click the Download button to download the converted animation as a TBA file.
